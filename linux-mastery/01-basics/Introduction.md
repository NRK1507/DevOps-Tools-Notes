# What is Linux?

Linux is an open-source operating system used in:
- Servers
- Cloud environments
- DevOps tools

## 🧱 Components of Linux

| Component | Description |
|----------|------------|
| Kernel | Core of OS |
| Shell | Command interface |
| File System | Stores data |

## 📁 Directory Structure

- `/` → Root
- `/home` → Users
- `/etc` → Config
- `/var` → Logs
- `/tmp` → Temporary files
